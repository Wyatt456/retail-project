# Retail ETL Project

This project performs data cleaning and exploratory analysis on the Online Retail II dataset.

## Project Structure

retail_project/
├── data
│   ├── cleaned
│   └── raw
├── notebooks
│   ├── 01-checkpoint.ipynb
│   └── 02_data_cleaning.ipynb
├── report
├── requirements.txt
├── src
│   └── download_data.py

## Dataset

Online Retail II Dataset from Kaggle.

Raw dataset is not stored in this repository.

## Setup

Create virtual environment:

python3 -m venv venv

Activate:

source venv/bin/activate

Install packages:

pip install -r requirements.txt
